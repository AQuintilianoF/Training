Python CLI Planner

This is a simple command-line planner application built in Python.

The purpose of this project is to practice object-oriented programming, clean architecture, and JSON data persistence.

Features:

Add tasks

View all tasks

Delete tasks

Store data in a JSON file

Project Structure:

main_planner.py → application entry point

ui_planner.py → command-line interface

service_planner.py → application logic

json_resp.py → JSON read/write operations

Classes.py → Event domain model

schedule.json → persistent storage

Concepts practiced:

Object-Oriented Programming

Class methods and properties

Layered architecture

Separation of responsibilities

File handling and JSON manipulation

How to run:

python main_planner.py

This repository represents my learning progress in Python and backend fundamentals.
